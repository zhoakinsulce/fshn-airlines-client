✈️ FSHN Airlines – Sistemi i Rezervimit & Check-in (Client)

Ky projekt përfaqëson pjesën Front-End të sistemit të rezervimit për FSHN Airlines. Aplikacioni është zhvilluar duke përdorur HTML5, CSS3 dhe JavaScript (ES6+), pa përdorimin e framework-eve si React apo Vue.

Klienti komunikon me një API Back-End për realizimin e funksioneve kryesore të sistemit, si:

kërkimi i fluturimeve,
rezervimi i biletave,
check-in online,
dhe gjenerimi i biletës elektronike.
🚀 Ekzekutimi i Projektit

Për funksionimin korrekt të aplikacionit, duhet të jetë aktiv edhe serveri i API-së.

1. Nisja e API-së (Back-End)

Hapni folderin e projektit FSHN-Airlines-API dhe ekzekutoni komandat:

npm install
npm start

Pas nisjes së serverit, API-ja do të jetë gati për të pranuar kërkesat nga klienti.
